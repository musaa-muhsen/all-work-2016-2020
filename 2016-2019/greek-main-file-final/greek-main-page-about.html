Greek Mythology Artifacts is a website based on well you guessed it Greek Mythology Artifacts. It’s designed for mobile/cell phone use, so the structure is contained in a rectangle shaped box representing the phone outline and under 500px the site changes to its natural structure. The whole website colour scheme is black and white, I used CSS GRID to make the lines that represent Greek style pillars. The site is a multiple page website (links below) which is something that I haven’t done for a while, personally I think multi page sites tend to create more depth to project work.
